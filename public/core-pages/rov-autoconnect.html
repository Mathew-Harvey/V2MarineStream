The Problem We Solved

Connecting ROVs to view live video feeds shouldn't be complicated

The Challenge

When connecting an ROV to a computer via network cable, users needed to manually determine the correct RTSP address to enter into VLC Media Player. This process was:

  • Time-consuming and technical
  • Prone to user error
  • Required networking knowledge
  • Caused delays in operations

Our Solution

ROV AutoConnect eliminates the guesswork by automatically:

  • Scanning network addresses via ping detection
  • Identifying the ROV's RTSP stream location
  • Launching VLC Media Player automatically
  • Connecting to the video feed seamlessly

Key Features

Designed for simplicity and reliability in the field

Automatic Detection

Intelligently scans your network to find the ROV's RTSP stream address without manual configuration.

VLC Integration

Seamlessly launches VLC Media Player and connects to the video stream in fullscreen mode for immediate viewing.

Modern Interface

Clean, intuitive design inspired by Google and Apple, featuring integrated MarineStream branding.

Progress Tracking

Real-time progress bar and status updates keep you informed during the connection process.

Customizable

Easily modify the POSSIBLE_RTSP_URLS list in the script to match your specific ROV setup requirements.

Open Source

Fully open-source code available on GitHub for transparency, customization, and community contributions.

How It Works

Simple, automated process gets you connected in seconds

1

Connect Your ROV

Connect your ROV to your computer via network cable and ensure it's powered on.

2

Launch ROV AutoConnect

Run the ROV AutoConnect executable. The modern interface will appear with connection options.

3

Automatic Detection

Click "Connect" and watch as the software automatically scans for and detects your ROV's RTSP stream.

4

Stream Live Video

VLC automatically launches in fullscreen mode, displaying your ROV's live video feed for immediate operation.
